## Key Components

### 1. Configuration Layer (`internal/config/config.go`)
- Loads `.env` file using `godotenv`
- Reads environment variables
- Provides `GetDatabaseDSN()` method
- Supports both individual params and complete URL

### 2. Database Layer (`internal/database/database.go`)
- Initializes GORM connection
- Configures connection pool
- Tests connection with ping
- Returns `*gorm.DB` instance

### 3. Main Application (`cmd/server/main.go`)
- Orchestrates startup sequence
- Handles errors gracefully
- Starts HTTP server

### 4. Test Utility (`cmd/test-db/main.go`)
- Validates database connectivity
- Displays diagnostic information
- Helps troubleshoot issues
